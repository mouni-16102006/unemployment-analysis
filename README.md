# 📊 Unemployment Analysis with Python - CodeAlpha Internship

## 📌 Project Overview
Analysis of unemployment trends in India using Python.
Built as part of the **CodeAlpha Machine Learning Internship**.

---

## 🎯 Objective
- Analyze unemployment rate data across Indian states
- Visualize trends over time
- Investigate the impact of Covid-19 on unemployment
- Identify seasonal patterns in the data

---

## 🛠️ Technologies Used
- Python 3.x
- Pandas
- Matplotlib
- Seaborn

---

## 📊 Dataset
- **Source:** Kaggle - Unemployment in India
- **Features:** State, Date, Unemployment Rate, Employed, Labour Rate
